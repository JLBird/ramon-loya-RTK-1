Executive Summary

RTK Security Labs conducted an independent adversarial validation of the A2SPA Protocol Control Layer using the RTK-1 autonomous red teaming platform. The engagement ran three attack scenarios — crescendo injection, unauthorized tool execution, and agentic chain exploitation — across two validation phases.

Phase 1 establishes the baseline Attack Success Rate without A2SPA active. This preliminary report presents Phase 1 findings. The binary C1/C2 verdict presented here reflects the target system's resilience under live adversarial conditions. Both C1 and C2 conditions held across all tested sequences.

Phase 2 will run identical attack sequences with A2SPA active. The ASR delta between Phase 1 and Phase 2 constitutes the primary quantitative proof of A2SPA's effectiveness for presentation to financial institution procurement officers.
